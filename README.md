# Customer Segmentation Dashboard

A modern customer segmentation and tag management dashboard built with React, TypeScript, GraphQL, Apollo Client, Node.js, and Tailwind CSS.

The application allows filtering customers, applying bulk tags, viewing customer history, and managing customer data through a clean admin dashboard UI inspired by CRM platforms like Shopify and HubSpot.

---

# Features

- Customer Filtering
- Bulk Tag Management
- Customer History Tracking
- Pagination
- GraphQL API
- Apollo Client Integration
- Modern Dashboard UI
- Tailwind CSS Styling
- TypeScript Support
